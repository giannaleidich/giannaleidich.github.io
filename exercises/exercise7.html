Cooking in Microsoft Word: A Beginner's Guide to Crafting Their Own Cookbook

This is a 20-30 page user manual for Microsoft Word. Specifically created for cookbook authors, Cooking in Microsoft Word guides users in writing and designing their own cookbook, from recipe layout to cover design. Developed over the course of a semester, this manual features a modular layout and an intuitive visual design and layout, and underwent several rounds of workshopping, usability testing, and revisions.

Initiative to Create Therapy Dog Program to Support Primary Care and Promote Animal-Assisted Therapy

This is a 20 page grant proposal seeking funding to create a therapy dog program at Putnam County Hospital. Initiative to Create Therapy Dog Program demonstrates expertise in argumentative and technical writing, as well as attention to detail when it comes to explanations, research, and structure.

UMass Housing-cyclopedia

This is a high-fidelity prototype for UMass Housing-cyclopedia, a website that would help users find on- and off-campus housing. Created in a team-based setting over the course of a semester, this prototype is the result of user research and testing, including identifying user tasks and end goals, low- and mid-fidelity prototyping and several usability studies and experiments. This website was developed with a human-first mindset.

View the Housing Website

Jabberwocky Instagram graphics

These are graphics created for the Jabberwocky—UMass Undergraduate Literary Journal—Instagram account to promote the journal. These graphics were designed to capture the attention of users by focusing on a minimalistic design that highlights the most important information: what pieces can be submitted and when the due date is.
